StudyBuddy is an AI-powered study app (like an AI Quizlet) built by Team16. Users upload course materials, and the AI generates flashcards, quizzes, and provides a chat tutor.

**Why:** Academic project to build an interactive AI study tool.

**How to apply:** All work should be in the context of a team project using TM16-xxx task IDs. The app has two main directories: `artifacts/frontend` (static HTML/JS) and `artifacts/backend` (FastAPI + OpenAI). Key gaps are: frontend-backend wiring, database persistence, backend auth, and testing.
