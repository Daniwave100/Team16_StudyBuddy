## Research on Website Acceptance Criteria
Web Content Accessibility Guidelines (WCAG) 2.1: 
https://www.w3.org/TR/2025/REC-WCAG21-20250506/

After reading these requirements, this is a very detailed description of website criteria that subsections perceivable, operable, understandable, robust, conformance.The guideline can be a great starting point for our website. To start, we will pick out a few of these requirements to meet for our StudyBuddy website and throughout the semester add more. Overall, we need to ensure that our website is readable, understandable, accessible, and operable.
